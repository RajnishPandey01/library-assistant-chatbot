# Library Assistant Chatbot Simple Full Stack Chatbot using: - FastAPI - SQLite - HTML/CSS/JS ## Setup Instructions 1. Open terminal 2. Go to backend folder cd backend 3. Install dependencies pip install -r requirements.txt 4. Run server uvicorn main:app --reload Server runs at: http://127.0.0.1:8000 Open frontend/index.html in browser. Test messages: - Is Python Basics available? - Who is the author of Rich Dad Poor Dad? - Show books in Programming category
